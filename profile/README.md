# YClawAI

**Your AI assistant just got a company to run.**

An open-source AI organization framework — 12 agents, 6 departments, multi-chain treasury, full governance — steered by YOUR AI assistant through 8 markdown files.

Extracted from a production system that ran 12 autonomous agents for over a year.

